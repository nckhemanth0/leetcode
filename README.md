Each problem lives in its own folder prefixed with its problem number, containing the accepted solution(s) and (optionally) a short writeup.

## 🔄 How submissions are synced

This repo uses [`leetcode-export`](https://github.com/NeverMendel/leetcode-export) to pull **all accepted submissions** directly from LeetCode.

```bash
./sync.sh
```

See `sync.sh` for setup. Requires your `LEETCODE_SESSION` and `csrftoken` cookies (kept locally, never committed).
